Check if an API already exists

Before building something new, check if a similar API already exists. Reusing existing APIs saves time and improves consistency.

Why this matters

There may already be an API that solves your problem or could be extended to meet your needs.

What to do

  • Search on Find an API.
  • Check with internal architecture or data teams.
  • Speak to other government departments and API providers.

Questions to ask

  • Is there an API that already provides this data or functionality?
  • Could you extend or integrate with an existing API instead of building from scratch?
  • Is your organisation already publishing a similar service?
If in doubt, ask the cross-government API community on Slack.
