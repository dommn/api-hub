Find the right API

Start by checking the GOV.UK API catalogue. You can filter by department, topic or access level, and search by keyword.

  • Read the API summaries to understand what each one does
  • Look for published documentation or OpenAPI specifications
  • Check if the API is open (public), partner-only, or internal

If nothing matches, speak to your digital team, ask the API Hub team, or post in the cross-government API Slack channel.
